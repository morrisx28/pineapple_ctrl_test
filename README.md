## Installation Guide
Follow install guide ref: [unitree_sdk2_python](https://github.com/unitreerobotics/unitree_sdk2_python.git)

## Quick start

# For user PC guide 
Once your computer connect the robot (connect by router)
```
python3 pineapple_test.py ur_internet_interface_name
```
For example
```
python3 pineapple_test.py enxcc96e5abe8c2
```

# For pineapple guide 
Remote to pineapple (ssh)
```
cd pineapple/build
sudo ./DDS_bridge enx00e09908c1ee
```
